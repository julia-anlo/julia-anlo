### 🌿 Obesity & Lifestyle Analytics
9-repo pipeline from text mining to predictive modeling on 2,111 lifestyle profiles.

| Project | Techniques | Key Result |
|---|---|---|
| [Text Mining](https://github.com/julia-anlo/text-mining-health-discourse) | NLP · Word Cloud | WHO frames obesity through behavior |
| [Correspondence Analysis](https://github.com/julia-anlo/correspondence-analysis-lifestyle) | CA · MCA | Transport mode predicts obesity class |
| [PCA](https://github.com/julia-anlo/pca-obesity-dimensionality) | PCA | 2 components · 41.3% variance |
| [Hierarchical Clustering](https://github.com/julia-anlo/hierarchical-clustering-obesity) | Ward · Manhattan | k=3 · 41.76% VE |
| [K-Means Segmentation](https://github.com/julia-anlo/kmeans-lifestyle-segmentation) | K-Means | 81.93% VE · standardization matters |
| [KNN Classification](https://github.com/julia-anlo/knn-obesity-classification) | KNN · Kernel | 87.07% accuracy · k=1 optimal |
| [Random Forest](https://github.com/julia-anlo/random-forest-obesity) | RF · Variable Importance | 96.37% · habits predict 81.55% alone |
| [SVM](https://github.com/julia-anlo/svm-multiclass-obesity) | Linear · RBF · Polynomial | 97.48% · linear wins |
| [Logistic Regression](https://github.com/julia-anlo/logistic-regression-multinomial) | Multinomial | 85.80% · fully interpretable |
